# JODOUR Website

JODOUR est une plateforme web dédiée à une marque de design marocain contemporain, combinant artisanat traditionnel et innovation moderne.

## Aperçu du projet

- Catalogue de produits
- Pages de collections
- Système d'authentification
- Panier d'achat
- Gestion des commandes
- Tableau de bord administrateur

## Structure du projet


jodour-web/
├── index.html
├── about/
├── collection/
├── products/
├── cart/
├── checkout/
├── auth/
├── admin/
├── assets/
└── _next/


## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- React
- Next.js



### Avec VS Code

- Installer l'extension Live Server.
- Ouvrir `index.html` avec Live Server.

## Fonctionnalités

### Utilisateurs

- Parcourir les collections
- Consulter les produits
- Créer un compte
- Se connecter
- Gérer le panier
- Passer une commande

### Administration

- Gestion des produits
- Gestion des commandes
- Gestion des utilisateurs
- Gestion des collections
